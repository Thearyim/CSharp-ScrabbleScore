## Description
**_This is an application that takes a word and returns the Scrabble score for that word. Here are the values of letters in this Scrabble:_**
* _Letters                          Value_
* _A, E, I, O, U, L, N, R, S, T       1_
* _D, G                               2_
* _B, C, M, P                         3_
* _F, H, V, W, Y                      4_
* _K                                  5_
* _J, X                               8_
* _Q, Z                               10_

| Example Input                                    | Example Output                                   |
| :----------------------------------------------- | :----------------------------------------------- |
| "BRYAN"                                          | "3+1+4+1+1 = 10"                                 |
